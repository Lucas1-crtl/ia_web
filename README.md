# 🤖 DeepSide - IA Local com Django

DeepSide é um chatbot simples desenvolvido em **Python e Django**, capaz de responder perguntas baseadas em intenções programadas.  
O projeto foi criado com o objetivo de praticar **desenvolvimento web, lógica de IA simples e manipulação de sessões no Django**.

---

## 🚀 Funcionalidades

- Chat em interface web
- Histórico de conversa
- Botão "Nova conversa"
- Respostas baseadas em intenções
- Normalização de texto (acentos e maiúsculas)
- Interface inspirada no ChatGPT

---

## 🛠 Tecnologias utilizadas

- Python
- Django
- HTML
- CSS
- SQLite (padrão do Django)

## ⚙ Como executar o projeto

Clone o repositório:
git clone : link do repositório

Entre na pasta :
cd nome da pasta .

Depois crie o ambiente virtual
python -m venv venv 

Ative o ambiente :
venv\Scripts\activate

Instale as dependências:
pip install django

Execute o servidor:
python manage.py runserver 

Abra no navegador:
http://127.0.0.1:8000
